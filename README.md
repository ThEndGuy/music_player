# Music player


Music player in C using raylib... I hate spotify...

This will be also shipping with an IDOC file which is an indentation based data document.

The .idoc files are parsed and interpreted by the C implementation in [idoc.c](./idoc/idoc.c).

An example on how such a file can be found in [test.idoc](./idoc/tests/test.idoc).


## Building the music player

To just build:
```console
.\nob
```
Build and run:
```console
.\nob run
```

## Building the idoc interpreter

To just build:
```console
.\nob
```
Build and run:
```console
.\nob idoc
```

(It will have command line arguments to choose the file, but not yet)
